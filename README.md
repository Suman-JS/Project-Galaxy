# Project-Galaxy
 A simple space game created using python and kivy

there is the instructions how to run the game on every platform

Windows :

Open cmd
Use git clone command
Cd to Project-Galaxy
Pip install kivy
Python main.py

Mac OS :

Open Terminal
Use git clone command
Cd to Project-Galaxy
Pip install kivy
Python main.py

Linux/Unix :

Open Terminal
Use git clone command
Cd to Project-Galaxy
Pip3 install kivy
Python3 main.py
