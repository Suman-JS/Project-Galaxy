# Project-Galaxy
 A simple space game created using python and kivy

## HOw to run the game

**STEP 1:**

Open cmd/terminal

**STEP 2:**

```bash
git clone https://github.com/Suman-JS/Project-Galaxy.git
```

**STEP 3:**

```bash
Cd Project-Galaxy
```

**STEP 4:**

```bash
pip install requirement.txt
```

**STEP 5:**

```bash
python main.py
```
